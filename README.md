# 🚗 Cars Sales Dashboard | Power BI

## 📌 Project Overview

The **Cars Sales Dashboard** is an interactive Business Intelligence solution developed in **Power BI** to analyze and visualize car sales data. The dashboard provides valuable insights into pricing trends, manufacturer performance, engine specifications, fuel types, and vehicle mileage, enabling users to make informed, data-driven decisions.

This project demonstrates the use of Power BI's data modeling, interactive filtering, and visualization capabilities to transform raw automotive sales data into meaningful business insights.

## 🎯 Objectives

* Analyze car prices across different manufacturers.
* Compare average vehicle prices and engine sizes.
* Monitor total mileage across the dataset.
* Explore sales information using dynamic filters.
* Build an interactive dashboard for efficient business reporting.

## 📊 Dashboard Features

* **Manufacturer-wise Sales Analysis** using interactive bar charts.
* **Average Car Price** KPI.
* **Average Engine Size** KPI.
* **Total Mileage** KPI.
* Dynamic slicers for:

  * Manufacturer
  * Fuel Type
  * Year of Manufacture
  * Model
* Clean and user-friendly dashboard design for easy navigation.

## 🛠 Tools & Technologies

* **Power BI Desktop**
* Power Query
* Data Modeling
* DAX Measures
* Interactive Visualizations

## 📈 Key Insights

* Compare total vehicle prices across major manufacturers.
* Identify pricing differences based on fuel type and manufacturer.
* Analyze average engine capacity and overall vehicle mileage.
* Filter the dashboard dynamically to explore specific manufacturers, models, and production years.
